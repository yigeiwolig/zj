# 安全加固使用示例

## 一、在云函数中使用频率限制

### 示例：在 `updatePageVisit` 云函数中添加频率限制

```javascript
// cloudfunctions/updatePageVisit/index.js
const cloud = require('wx-server-sdk');
cloud.init({ env: cloud.DYNAMIC_CURRENT_ENV });
const db = cloud.database();

exports.main = async (event, context) => {
  const wxContext = cloud.getWXContext();
  const openid = wxContext.OPENID;
  
  // 🔴 第一步：检查频率限制
  const rateLimitRes = await cloud.callFunction({
    name: 'rateLimit',
    data: {
      action: 'update_page_visit',
      maxCount: 30, // 1分钟内最多30次
      windowMs: 60000 // 1分钟
    }
  });
  
  if (!rateLimitRes.result.success) {
    return {
      success: false,
      error: rateLimitRes.result.error,
      retryAfter: rateLimitRes.result.retryAfter
    };
  }
  
  // 🔴 第二步：正常业务逻辑
  const { pageRoute } = event;
  
  // ... 原有的业务逻辑 ...
  
  return { success: true };
};
```

## 二、在小程序端调用带频率限制的云函数

### 示例：在 `app.js` 中调用

```javascript
// miniprogram/app.js
updatePageVisit: function(pageRoute) {
  // 异步调用，不阻塞页面加载
  wx.cloud.callFunction({
    name: 'updatePageVisit',
    data: { pageRoute: pageRoute },
    success: (res) => {
      if (res.result.success) {
        console.log('[app] 页面访问统计更新成功:', pageRoute);
      } else if (res.result.error === '请求过于频繁，请稍后再试') {
        console.warn('[app] 请求过于频繁，已跳过本次统计');
        // 可以选择延迟重试或直接忽略
      }
    },
    fail: (err) => {
      console.error('[app] 页面访问统计更新失败:', pageRoute, err);
    }
  });
}
```

## 三、数据库权限设置步骤

### 1. 打开数据库权限设置
1. 打开微信开发者工具
2. 点击"云开发" → "数据库"
3. 选择要设置的集合（如 `shouhou_repair`）
4. 点击"权限设置"

### 2. 设置权限规则

**方式一：使用预设规则**
- 选择"仅创建者可读写"（最简单，推荐）

**方式二：自定义规则（更灵活）**

```json
{
  "read": "doc._openid == auth.openid || doc.openid == auth.openid",
  "write": "doc._openid == auth.openid || doc.openid == auth.openid"
}
```

**管理员集合权限（guanliyuan）：**
```json
{
  "read": "auth.openid in get('/database/guanliyuan').data.map(d => d.openid || d._openid)",
  "write": false
}
```

### 3. 需要设置权限的关键集合

| 集合名称 | 推荐权限 | 说明 |
|---------|---------|------|
| `sn` | 仅创建者可读写 | 设备绑定信息 |
| `shouhou_repair` | 仅创建者可读写 | 售后维修单 |
| `my_read` | 仅创建者可读写 | 设备绑定申请 |
| `video` | 仅创建者可读写 | 视频案例 |
| `shop_orders` | 仅创建者可读写 | 订单信息 |
| `guanliyuan` | 仅管理员可读 | 管理员列表 |
| `login_logbutton` | 仅云函数可读写 | 封禁控制 |
| `valid_users` | 仅云函数可读写 | 白名单 |

## 五、部署频率限制云函数

### 1. 部署步骤

1. 在微信开发者工具中，右键点击 `cloudfunctions/rateLimit` 文件夹
2. 选择"上传并部署：云端安装依赖"
3. 等待部署完成

### 2. 创建数据库集合

1. 打开云开发 → 数据库
2. 创建集合 `rate_limit_logs`
3. 设置权限为"仅创建者可读写"（或"仅云函数可读写"）

## 八、测试频率限制

### 测试脚本（在小程序中）

```javascript
// 测试频率限制
async function testRateLimit() {
  for (let i = 0; i < 15; i++) {
    const res = await wx.cloud.callFunction({
      name: 'rateLimit',
      data: {
        action: 'test',
        maxCount: 10,
        windowMs: 60000
      }
    });
    console.log(`第 ${i + 1} 次请求:`, res.result);
    if (!res.result.success) {
      console.log('频率限制生效！');
      break;
    }
  }
}
```

---

**重要提示**：
1. 数据库权限是最重要的安全措施，必须立即设置
2. 频率限制可以有效防止爬虫，建议在所有关键操作中添加
3. 环境变量比代码中的常量更安全，建议迁移敏感信息
