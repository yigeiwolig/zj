# 上传照片文件夹

这个文件夹用于在**微信开发者工具**中手动上传产品照片。

## 使用说明

### 在微信开发者工具中上传图片：

1. 打开微信开发者工具
2. 在左侧文件树中找到 `miniprogram/uploads/images/` 文件夹
3. 将图片文件直接拖拽到这个文件夹中，或者右键选择"上传文件"
4. 图片上传后，可以在小程序中通过代码读取这些图片路径

### 在小程序中预览：

- 小程序中只提供预览功能，不进行实际上传
- 用户可以选择图片进行预览效果展示
- 实际图片文件需要在开发者工具中手动上传

## 文件结构

```
uploads/
  └── images/          # 图片存储目录（在开发者工具中上传）
      └── README.md    # 说明文件
```

## 注意事项

- 图片文件建议使用 `.jpg` 或 `.png` 格式
- 建议压缩图片以节省空间
- 在小程序中读取图片时，需要使用相对路径，例如：`/uploads/images/photo1.jpg`
